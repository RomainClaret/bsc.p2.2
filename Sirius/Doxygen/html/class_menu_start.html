Detailed Description

Main menu, which appears at the game startup.

It allows the user to select or create a profil and run a game. Games are saved in a json file type. Saves are found in: MacOSX : "Game.app/Contents/MacOS/save.json", Windows : Dans le meme dossier que Game.exe. Save files are not encrypted.
